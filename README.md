# QResolve

> AI-powered Support Ticket Classification and Prioritization Platform

## Project Vision

QResolve helps organizations automatically:

- Classify support tickets
- Predict ticket priority
- Route tickets to the correct department
- Improve customer response times

## Tech Stack

- Python 3.11
- PyCharm
- Git & GitHub
- Pandas
- NumPy
- Scikit-learn
- FastAPI
- Docker
